C API Documentation

The Python API has two abstraction levels.

  • Python-Basic-API provides the RAW_VNA class, and is a thin wrapper around the underlying c API. It handles type-conversion, keeping track of the task handle, and little else.

  • Python-OOP-API provides the VNA class, and is a higher-level interface that makes common VNA tasks easier to implement. It is still, to some extent, a work-in-progress.
